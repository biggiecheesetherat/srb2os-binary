# Sonic Robo Blast 2 (SRB2OS Edition)
## Code notice
The majority of the code is owned by Sonic Team Junior. I only made patches in the code for it to be usable in SRB2OS. I've linked the original repo below too for you to see. If you want to see the source code for the OS, you can download the ISO and decompress the squashfs inside said ISO using this guide (https://askubuntu.com/questions/437880/extract-a-squashfs-to-an-existing-directory/437883#437883). Also I apologize for the AI-like code, it's how I write C code as of now and I'm getting better at making it less robotic.

[Click here to download SRB2OS v0.1 (4GB) ](https://pixeldrain.com/u/9TwRAdg9)

[Click here to see the original Git repo](https://github.com/STjr/SRB2)

[Sonic Robo Blast 2](https://srb2.org/) is a 3D Sonic the Hedgehog fangame based on a modified version of [Doom Legacy](http://doomlegacy.sourceforge.net/).

## Dependencies
- SDL2
- SDL2-Mixer
- libupnp
- libgme
- libopenmpt
